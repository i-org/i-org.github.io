2.4.8.4.1 Examples of Sharpsign Asterisk

For example, 

  #*101111
 #6*101111
 #6*101
 #6*1011
all mean the same thing: a vector of length 6 with elements 1, 0, 1, 1, 1, and 1.

For example: 

 #*         ;An empty bit-vector
