Standard Generic Function (SETF CLASS-NAME)

●语法设定:

(setf class-name) new-value class => new-value

Method Signatures:

(setf class-name) new-value (class class)

●参数和值:

new-value---a symbol.

class---a class.

●详情:

The generic function (setf class-name) sets the name of a class object.

●例子: None.

●受制于： 无。

●例外情况： 无。
