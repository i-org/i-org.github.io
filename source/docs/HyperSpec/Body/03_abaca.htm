3.1.2.1.3.1 Examples of Self-Evaluating Objects

Numbers, pathnames, and arrays are examples of self-evaluating objects. 

 3 =>  3
 #c(2/3 5/8) =>  #C(2/3 5/8)
 #p"S:[BILL]OTHELLO.TXT" =>  #P"S:[BILL]OTHELLO.TXT"
 #(a b c) =>  #(A B C)
 "fred smith" =>  "fred smith"
